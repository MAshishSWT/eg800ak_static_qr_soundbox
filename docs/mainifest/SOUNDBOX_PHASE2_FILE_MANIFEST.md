# EG800AK Static QR UPI Soundbox Phase 2 Package Manifest

This package extends the approved Phase 1 soundbox skeleton with the KAE8_SQ1 BSP/HAL layer for GPIO, LED, keys, battery ADC, and speaker amplifier shutdown control.

The Quectel SDK common headers and libraries are preserved. The root Makefile builds only the production soundbox application module under `interface/soundbox`.

See `docs/soundbox_phase2/PHASE2_FILE_MANIFEST.md` for added and modified files.
